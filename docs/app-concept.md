# Assignment 2.1: Konzept für JavaScript-Erweiterung

Die JavaScript-Anwendung des Projekts soll aus vier kleineren Anwendungen bestehen, die NutzerInnen beim Organisieren und Verfolgen ihrer Projekte unterstützen. Geplant sind dabei folgende Funktionen: Projekte anlegen und anzeigen, eine Checkliste für einzelne Arbeitsschritte, eine Fortschrittsanzeige sowie einen Maschen- und Reihenzähler. Über das bereits vorhandene HTML-Formular können neue Projekte angelegt werden. Die eingegebenen Daten sollen anschließend mit JavaScript verarbeitet und als Projektkarte im Dashboard angezeigt werden. NutzerInnen sollen zudem eigene Checklistenpunkte hinzufügen und erledigte Arbeitsschritte abhaken können. Aus den als erledigt markierten Checklistenpunkten soll der Fortschritt berechnet und direkt auf der jeweiligen Projektkarte im Dashboard dargestellt werden. Zusätzlich ist ein Maschen- bzw. Reihenzähler geplant. Dabei soll es möglich sein, über Buttons (Reset, -1, +1, +5, +10) mit dem Zähler zu interagieren und so den aktuellen Stand zu erfassen. Die Anwendung soll sich dabei direkt in die bestehende Website einfügen. Das Dashboard bietet eine Übersicht über alle Projekte, während auf den einzelnen Projektseiten die Checklisten- und Zählerfunktion genutzt werden können. Die Einbindungsstellen in HTML bestehen bereits, sind derzeit jedoch noch statisch umgesetzt, so z.B. die Projektkarten oder die Checkliste. Diese Funktionen stellen die geplanten Grundanwendungen dar, wobei noch nicht sicher ist, ob diese gegebenenfalls für das Assignment reduziert werden sollten.

Für Assignment 2 soll der Fokus zunächst auf der Formularfunktion liegen. Das bestehende HTML-Formular soll mit JavaScript erweitert werden, sodass neue Projekte interaktiv erstellt, im LocalStorage gespeichert und anschließend dynamisch als Projektkarten auf der Startseite angezeigt werden können.
